# Festival DevOps Music Fest

Este es mi proyecto para la actividad de aprendizaje 6 del programa de DevOps y Contenedores. Aquí organicé toda la estructura del festival usando el flujo de Git.

## Tecnologías del Proyecto
* **Frontend:** Carpeta con los archivos de diseño básicos (index.html y style.css).
* **Backend:** Carpeta configurada para una API en Python usando Flask.
* **Docker:** Archivos Dockerfile y docker-compose.yml listos para montar los contenedores del entorno.
* **Control de Versiones:** Uso de Git local trabajando con ramas por funcionalidad y repositorio remoto conectado a GitHub[cite: 1, 2].
